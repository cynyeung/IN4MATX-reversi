About

Reversi is a strategy based game for two players, played on an 8×8 uncheckered board. Players take alternating turns placing their assigned color of light or dark game pieces called disks or tokens on the board. During a player's turn, the objective is to capture as many of your opponent's tokens through a variety of directional legal moves between two of the active player's tokens. The player with the most tokens on the board at the end of the last playable move wins the game.

This digital version of the game Reversi is a node.js project using HTML, CSS, and jQuery. The project source code is hosted on Github and the application is deployed on Heroku. This application utilizes Bootstrap, a CSS Framework for its responsive layout and components such as buttons and progress bar. This project also uses Socket.io, a Javascript library that enables real-time communication between the web client and server. This is demonstrated with the player connections in the game lobby, through the real-time messaging feature of the application, and the live play of tokens on the game board.

For the design approach, it is a unique stylistic choice loosely based around an 8-bit aesthetic. A custom font called VT323 from Google Fonts was selected to pair with the visual aesthetic of the game. Custom CSS was written in addition to the Bootstrap framework to style the game cohesively to the design intentions such as background images and consistent visual components across the pages.

Future iterations of this game could revisit the use of pngs for the tokens and alternatively use SVGs and CSS animations for the graphical elements. This web application could also review and build a more refined and robust mobile experience for the web application—Specifically as it relates to the hover state or game board hints of playable moves. Lastly, all the colors used throughout the game should be verified as WCAG compliant for web accessiblity.

Regarding the scalability and maintanence of the source code, this project could use a templating language such as Handlebars to DRY up semantic and structural code. A reusable global project stylesheet could also be introduced to help maintain CSS style consistency across the pages of the web application.

For additions, this version of the game could be a more immersive experience by introducing sound design to the animations of the tokens.
